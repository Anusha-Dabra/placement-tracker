<h1>PlacementPal</h1>

A simple web app to track internship and placement applications — built for college students.

Add a company, set a deadline, and get a reminder the day before it closes.



<h2>Features</h2>
<ul>
  <li>Add companies with just a name and deadline</li>
  <li>One-tap status updates (Interested → Applied → Interview → Offer → Rejected)</li>
  <li>Auto priority based on how close the deadline is</li>
  <li>Browser notification 1 day before deadline</li>
  <li>All data saved locally in your browser — no login needed</li>
  <li>Multiple users can use the same link with separate data</li>
</ul>

<h2>How to Use</h2>
Just open the link, enter your name, and start adding companies.

https://placement-tracker-ruby.vercel.app/

<h2>Run Locally</h2>
Download the files and open `index.html` in your browser. That's it.

<h2>Tech</h2>
HTML, CSS, and JavaScript
